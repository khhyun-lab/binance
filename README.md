# Binance Futures Bot

바이낸스 USD-M 선물 자동매매 봇입니다.

구성 요소:

- 바이낸스 선물 REST 기반 주문/계좌 조회
- 1분, 3분, 5분, 15분 컨텍스트 기반 진입 판단
- 손절 대비 2:1 익절 비율 기반 청산 모델
- heartbeat 파일 기반 단일 watchdog 재시작
- 텔레그램 알림
- macOS launchd 예시 포함

## 설치

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install -e .
```

## 실행

봇 단독 실행:

```bash
python3 -m binance_bot.main
```

Binance 선물 봇 1회 점검 실행:

```bash
python3 -m binance_bot.main --once
```

Binance 선물 실거래 사전점검:

```bash
python3 -m binance_bot.main --preflight
```

사전점검은 잔고 조회, 심볼별 현재 설정 레버리지 적용, 롱/숏 테스트 주문 전송까지 수행하며 실제 체결은 발생시키지 않습니다.

## launchd 예시

실사용 plist는 deploy/com.binance.bot.watchdog.plist 이고, 설치/재시작은 scripts/watchdog_service.sh 로 처리합니다.

```bash
./scripts/watchdog_service.sh install
./scripts/watchdog_service.sh status
```

이미 등록된 watchdog을 다시 올릴 때는 아래를 사용합니다.

```bash
./scripts/watchdog_service.sh restart
```

## 백테스트 보조 스크립트

```bash
python scripts/analyze_decisions.py reports/backtests/20260508T000000Z
python scripts/compare_backtests.py reports/backtests/run_a reports/backtests/run_b
python scripts/run_strategy_experiments.py --symbols BTCUSDT ETHUSDT --start 2026-04-01 --end 2026-04-30
```

- analyze_decisions.py: decision_log.jsonl의 hold 이유, candidate 유형, blocker 분포를 요약한다.
- compare_backtests.py: 여러 report 디렉터리의 summary.json, trades.csv를 읽어 순손익, MDD, 진입 유형/청산 사유 분포를 비교한다.
- run_strategy_experiments.py: baseline, conservative, balanced, aggressive pullback variant를 묶어서 실행한다.
