# Binance Futures Bot

바이낸스 USD-M 선물 자동매매 봇만 분리한 저장소입니다.

이 저장소에는 다음만 포함합니다.

- `binance_bot/`: 바이낸스 선물 전략, 상태 저장, REST 서비스, 앱 진입점
- `upbit_bot/heartbeat.py`: 하트비트 기록 유틸
- `upbit_bot/logging_utils.py`: 로그 설정 유틸
- `upbit_bot/telegram.py`: 텔레그램 알림 유틸

업비트 현물 봇, 시장 스캐너, 혼합 watchdog 코드는 포함하지 않았습니다.

## 요구 사항

- Python 3.10+
- 바이낸스 USD-M 선물 API Key / Secret

## 설치

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
```
